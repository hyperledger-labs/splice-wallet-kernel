# portfolio

A example dApp that imports the dApp SDK and communicates with a Wallet Gateway.
This app aimes to provide a minimal but complete portfolio, using React+TS Vite.

## Building and running

Because this example has a number of dependencies, we recommend using the
scripts in the root repository for development.

- Build: `yarn build:all`
- Start all services (including this one): `yarn start:all`
    - This service lives at <http://localhost:8081>
- Stop all services again: `yarn stop:all`
